# knit-result-download
This is the script which is written in python to download the all result data and store in the sqlite database and also print in the screen.
first thing we have to run rollno.py, it will create a roll_list.txt which contain all roll number of knit student.
Then we have to run knit_Result_fetching_program.py which will create a file result_2017.sqlite and folder named 'pdf' and then inside the folder it create a pdf copy of the result.
